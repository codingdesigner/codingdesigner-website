So <pre>&#x3C;i class=&#x22;icon--search&#x22;&#x3E;Search&#x3C;/i&#x3E;</pre> becomes <i class="icon--search">Search</i></p> <p><strong>In Sass, as a psuedo selector</strong></p> <p>Use the mixin `icon` with the argument of the icon name like this: `@include icon(facebook);`. So</p> <p>HTML</p> <pre>&#x3C;div class=&#x22;demo-random-icon&#x22;&#x3E;Random icon here&#x3C;/div&#x3E;</pre> <p>CSS</p> <pre>.demo-random-icon { @include icon(facebook); }</pre> <p>becomes</p> <div class="demo-random-icon">Random icon here.</div> <p><strong>In Sass, as a text-replacement</strong></p> <p>This may be more familiar for those coming from image sprites. Use the mixin `icon` with the argument of the icon name plus "$text-replace: true" as an argument: `@include icon(twitter, $text-replace: true);`. So</p> <p>HTML</p> <pre>&#x3C;div class=&#x22;demo-text-replace-icon&#x22;&#x3E;This text will be gone&#x3C;/div&#x3E;</pre> <p>CSS</p> <pre>.demo-text-replace-icon { @include icon('twitter', $text-replace: true); height: 16px; width: 16px; }</pre> becomes <div class="demo-text-replace-icon">This text will be gone</div> <p>You're responsible for height, width, positioning, etc.</p> <p><strong>Adding and generating icons</strong></p> <p>SVG files added to `source/images/icons/src/` while running `grunt` are automatically added to the font face.
